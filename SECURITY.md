# Security Policy

## Repository archive status

This repository is archived and no longer actively maintained.
It remains public only for study purposes.
There is no guarantee of response, review, or fix, including for security reports.

## Supported versions

| Version | Supported |
| --- | --- |
| 2.x | Yes |
| 1.x | No |

## Reporting a vulnerability

Please report vulnerabilities privately by emailing security@botinho.dev.

Include:
- Affected endpoint or module
- Reproduction steps
- Expected and actual behavior
- Suggested mitigation if available

## Response SLA

- No response SLA is guaranteed for this archived repository.

## Disclosure policy

- Do not open public issues for security reports.
- Coordinated disclosure is preferred after patch release.
